## → Make it an editable Google Slides (team can edit)

PowerPoint imports into Google Slides natively and stays fully editable:

1. Go to **drive.google.com** (or **slides.google.com**).
2. **Drag `child-safety-chatbot.pptx` into the browser** (or *New → File upload*).
3. Right-click the uploaded file → **Open with → Google Slides**.
   (Or, once open, **File → Save as Google Slides** to get a native copy.)
4. **Share** (top-right) → add your teammates with **Editor** access, or set
   “Anyone with the link → Editor”. Now everyone can edit.

That's it — one drag + one share. The fonts/colors/screenshots come across.

> Why not auto-uploaded for you: the Drive integration only accepts inline file
> bytes, and a 160 KB binary is too large to push that way — so the import is
> done from your own Google account (which also makes you the owner).
